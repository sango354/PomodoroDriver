## Repository Rules

- Do not inspect Git status, diffs, branches, logs, staged files, or commits unless the user explicitly asks for a Git-related action.
- Do not run Git commands unless the user explicitly asks for them.
- Do not mention Git state in progress updates or final responses unless it is directly relevant to an explicit user request.
- Continue to avoid overwriting or reverting unrelated user changes while editing files.
