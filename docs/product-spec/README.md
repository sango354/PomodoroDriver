# 產品規格索引

本目錄存放 Lo-fi 專注陪伴產品的工作中規格文件。

## 目錄結構

- `PRD.md`：產品需求文件
- `systems/`：系統層級規格
- `data/schema.md`：邏輯資料結構
- `engineering/roadmap-and-backlog.md`：開發階段與工作拆解

## 產品摘要

- 產品類型：Lo-fi 專注陪伴產品
- 核心主張：結合實用的專注工具、輕度情感陪伴與可見的進度回饋
- MVP 範圍：單一陪伴角色、專注計時器、任務清單、回饋系統、解鎖內容、輕度情境變化
